Name Description Syntax Example
Flash Flash button for given amout of time flash(id, duration) flash("row1item1", 4000);
Background Flash Flash window background for given amout of time and color bg_Flash(color, duration) bg_Flash("Red", 4000);
Light Up Light Up the Button until sample isn't finished
Note that color argument is optional.		 lightUp(id, color) lightUp("row1item1", "Red");
Light Up Background Light Up the background until sample isn't finished
Note that color argument is optional.		 bg_lightUp(id, color) bg_lightUp("row1item1", "Red");
